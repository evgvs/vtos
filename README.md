# vtOS - open source operating system 

## Build & run

### Enter build environment

```
$ ./env.sh
(vtos_env) $ make qemu # build and test in a qemu vm
```

### Make an ISO file

```
$ make iso
```

## Features
*  Kernel
*  Print color text on screen
*  Keyboard 
*  Ports IO
*  Panic
*  Shell
*  GRUB boot support
